Recipe suggestions are displayed for users' convenience.

{% trans "Recipe Menu" %}

The recipe menu provides preconfigured recipe keyword listings along with options to create your own recipes.

{% trans "Create Recipe" %}

Select the New option from the recipe menu. Enter the basic recipe and save, you'll be directed to the recipe page so that ingredient and instructions can be added.

{% trans "Prep and Cook Time Entry" %}

Recipe preparation and cook times may be entered using the following syntax:

    {% for duration in duration_help %}
  • {{ duration.symbol | safe }} - {{ duration.period | safe }} : {{ duration.enter_as | safe }}
    • {% endfor %}

The order is not important.

{% trans "Search" %}

{% trans "Basic Search" %}

Enter a keyword in the search box in the navigation bar at the top of the screen, and click Search to find recipes containing the keyword.

{% trans "Category Search" %}

From the navigation bar select the All option to get a list of all categories in alphabetical order.
